## 工作表：Sheet1
1: 论文写作中非学术性错误的主要表现
2: 类别 | 细目 | 主要表现 | 常见问题描述
3: 基本语言规范 | 字词 | ◆错字 ◆别字 ◆漏字 ◆赘字 | ▲“的”、“地”、“得”不分。如将“为了更好地验证本算法的效果”误为“为了更好的验证本算法的效果”。
4:  |  |  | ▲拼音等输入法录入错误同音或同形词。
5:  |  |  | ▲将学位论文翻译为 paper（应为thesis）。
6:  |  |  | ▲常识错误。比如把地区称为国家。
7:  |  |  | ▲数字中的小数点错误。
8:  | 句法 | ◆病句 ◆错句 ◆中文式英文 | ▲句子缺乏主语。比如用短语当句子。
9:  |  |  | ▲一句话内前后主语不一致，逻辑混乱，可读性差。
10:  |  |  | ▲按中文习惯翻译英文摘要。比如直接在英文摘要中用paper 等当主语。
11:  |  |  | ▲用机器软件翻译中文摘要导致的各种字、词、句低级错误。
12:  | 标点符号 | ◆该用不用 ◆乱用 ◆误用 | ▲不断句。比如一长段话仅一个句号。
13:  |  |  | ▲引用他人文字没有使用引号（将导致学术不当行为）。
14:  |  |  | ▲句后缺乏句号。
15:  |  |  | ▲括号、引号等符号没有成对使用，正确配套。
16:  |  |  | ▲用“如下”等引出下文时缺乏冒号。
17:  |  |  | ▲乱用符号。比如在句字中随意打逗号分割断句，或该用句号的用逗号。
18:  |  |  | ▲破折号“——”与连接号“－”混用。
19:  | 语种 | ◆中西文语言错误混用 ◆中西文符号错误混用 | ▲中文论文中提供的图、表、公式中错误出现英文文字。
20:  |  |  | ▲中英文标点符号错误混用。如英文摘要中出现中文逗号；公式编号等处的阿拉伯数字编号用中文括号配套。
21: 通用内容规范 | 一致性 | ◆前后内容不一致 | ▲多处出现的对象、变量、名称等前后不统一（包括同一参数在图、表中与在正文中的不一致，同一参数大小写的不统一）。
22:  |  |  | ▲中西文摘要的内容、关键词等不一致。
23:  | 完备性 | ◆文档结构缺项 ◆内容缺乏 | ▲中文摘要后缺乏关键词等文档要素。
24:  |  |  | ▲首次出现变量、缩写、专业名词等未说明含义。
25:  |  |  | ▲摘要、正文的内容分别不能自解释。比如摘要中含引用论文编号或变量名、缩写等不明信息。
26:  |  |  | ▲变量、数字出现时缺乏名称、量纲说明。
27:  |  |  | ▲出现图、表时缺乏相应说明。
28: 通用内容规范 | 相关性 | ◆出现无关内容 ◆内容和形式不相关 | ▲有隐藏的无关信息。
29:  |  |  | ▲文本内容被故意图形化。
30:  |  |  | ▲图、表混用。应用表说明的误用图，或表目与表身分界不清晰。
31:  | 非必要 冗余 | ◆内容多处重复 | ▲一段文字或句子在摘要、正文等处出现不止一次。
32: 专业写作规范 | 文字 | ◆口语化 ◆不遵循行业惯用语要求 ◆违反保密等规定 | ▲写作中使用口语。
33:  |  |  | ▲学位论文摘要、正文等地用“我们”或“we”当主语。
34:  |  |  | ▲随意更改行业惯用语。如用小写英文字母表示行业约定的缩写。
35:  |  |  | ▲涉及隐私、保密等专业要求时未按规定处理。如保密项目名称直接明文出现。
36:  |  |  | ▲数量转行。如同一数量，一部分在上一行，部分在下一行。
37:  |  |  | ▲公式转行不规范。公式通常在加、减、乘运算符号处转行，如果是乘运算应在上一行末加点乘号。如果是矢量运算应在合适的地方转行或附加说明。
38:  | 图表和 公式 | ◆类型混乱 ◆内容缺乏 ◆质量差 ◆编号错误 ◆放置位置错误 | ▲未按行业规范的图形类型制图。如功能模块图混合系统流程图，动态、静态图不分；程序流程图出口众多。
39:  |  |  | ▲随意编造图形元素，错误制图。如箭头流转混乱；图上的线条无法区分，随意交叉。
40:  |  |  | ▲图元素缺乏。如判断框分支标识不全。
41:  |  |  | ▲未按要求插入矢量图，截图质量差，不清晰。
42:  |  |  | ▲有错误或多余信息。如从 word中截取图表，图上还有回车符号。
43:  |  |  | ▲无坐标单位，无图例。
44:  |  |  | ▲图、表格式、风格不一致，比如字体、字号、线条粗细等。
45:  |  |  | ▲图中字太小，难以辨认。
46:  |  |  | ▲图名在图之前，或图出现在正文说明之前，甚至缺少相应图、表说明。
47:  |  |  | ▲表名在表之后。
48:  |  |  | ▲公式编号混乱，或编号中出现“公式”字样。
49:  | 摘要和 结论 | ◆不规范 ◆总结归纳不够 | ▲摘要写作不规范，不能对论文进行高度概括。
50:  |  |  | ▲没有对取得的成果进行提炼，仅仅是重复描述所做的工作，没有与绪论相呼应。
51: 专业写作规范 | 参考文献 | ◆格式错误 ◆要素缺乏 | ▲未按相关的规范整理格式。
52:  |  |  | ▲作者姓和名顺序混乱（应姓在前，名在后），未按要求缩写其名、缩写名未大写、或乱加缩写点（应不加）。
53:  |  |  | ▲缺乏卷期、出版地、页码、文献类型等要素。
54:  |  |  | ▲拼音、西文中的大小写、标点符号未按要求统一。
55:  |  |  | ▲多处格式细节不统一（并非对标准理解不同导致）。
56:  |  |  | ▲文后参考文献未按照在文中引用的先后顺序排列。
57:  | 引用 | ◆引用错误 ◆引用不当 ◆该引不引 | ▲章节标题上加引用符号。
58:  |  |  | ▲使用他人的公式、图、表未明确给出引用出处（将导致学术不当行为）。
59:  |  |  | ▲使用相对引用时出错。如“见下图”时，文字下并没有图，已经被排版到别的地方。
60:  |  |  | ▲正文中参考文献、公式、图表等的引用关系混乱。如错位引用。
61:  |  |  | ▲正文中提及参考文献作者时，未按原语种惯例排列其姓和名的顺序（中文应姓前名后，英文反之），西文缩写名的字母未大写或其后未加缩写点。
62: 编辑出版规范 | 模版 | ◆不符合标准格式 ◆格式不统一 ◆格式错误 | ▲随意更改全文字体、字号、行距、对齐方式等。
63:  |  |  | ▲全文各处格式不一致。如一个图、表内的字号、字体不统一，忽大忽小；如一个变量在正文里以公示出现时随意缩放大小，明确区别于正文文字，或影响正常行距。
64:  |  |  | ▲变量等格式错误。如正体、斜体、粗体等未正确使用
65:  |  |  | ▲（如向量、矩阵未用粗斜体）；变量下标未编辑为下标； 变量无量纲、无单位或单位错误。
66:  |  |  | ▲章节、图表、公式、参考文献等未按要求编号。如学位论文中公式、图表等要求按章独立编号。
67:  |  |  | ▲公式、图表的编号不连续。
68:  |  |  | ▲格式细节错误。如公式编号未右对齐，同时公式本身居中；要求居中的标题等内容而未居中；在设置变量斜体、粗体时把前后括号、标点符号一律包含在内。
69:  |  |  | ▲参考文献格式不按模板要求，信息不全。
70:  | 版面 | ◆版面混乱 ◆行距不一 | ▲文中随意“开天窗”，出现不合理的空白行（如 3-5行以上）并严重影响版面完整性。
71:  |  |  | ▲正文和引用的图表距离太远。
72:  |  |  | ▲图、表的标题和其内容分离在不同页面，被粗暴截断。
73:  |  |  | ▲标题出现在页面的最后一行。
